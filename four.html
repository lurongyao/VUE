手动配置自己:

webpack+vue-loader



webpack加载模块

如何运行此项目？

1. npm install    或者    cnpm install

2. npm run dev

    -&gt;  package.json

        "scripts":{

            "dev":"webpack-dev-server --inline --hot --port 8082"

        }

以后下载模块：

npm install &lt;package-name&gt;  --save-dev

EADDRINUSE 端口被占用

少了:

webpack-dev-server

webpack

.vue 结尾，之后称呼组件

路由:

vue-router



    -&gt;  如何查看版本:

        bower info vue-router



路由使用版本: 0.7.13

配合vue-loader使用:

  1. 下载vue-router模块

    cnpm install vue-router@0.7.13

  2. import VueRouter from 'vue-router'

  1. Vue.use(VueRouter);

  1. 配置路由

    var router=new VueRouter();

    router.map({

     路由规则

    })

  2. 开启

    router.start(App,'#app');

注意:

之前: index.html    -&gt;   &lt;app&gt;&lt;/app&gt;

现在: index.html    -&gt;   &lt;div id="app"&gt;&lt;/div&gt;



App.vue    -&gt;   需要一个 &lt;div id="app"&gt;&lt;/div&gt;  根元素
